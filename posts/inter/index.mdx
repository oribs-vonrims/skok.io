Check out all Inter var features on the [lab page](https://rsms.me/inter/lab/?feat-case=1&invert-colors=1&varWeight=500&varfont=1)

- `U+20-7E` - [Basic Latin](https://en.wikipedia.org/wiki/List_of_Unicode_characters)
- `U+00A9` - `©`
- `U+2026` - elepsis `…`
- `U+A` - newline
- `U+00A0` - Non-breaking space

1. Critical - only `U+20-7E`
2. Regular / _Italic_

```
pyftsubset Inter.otf \
  --unicodes="U+20-7E,U+00A9,U+2026,U+A,U+00A0" \
  --layout-features="ss01,ss02,ss03,kern,calt" \
  --flavor="woff2" \
  --output-file="inter-var-italic-subset.woff2"
```
