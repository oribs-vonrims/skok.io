```
ABCDEFGHIJKLMNOPQRSTUVWXYZ
abcdefghijklmnopqrstuvwxyz
1234567890
`~!@#$%^&*()<>_-=+|\/[]{}'":;.,
```

- `U+20-7E` - [Basic Latin](https://en.wikipedia.org/wiki/List_of_Unicode_characters)
- `U+00A9` - `©`
- `U+2026` - elepsis `…`
- `U+A` - newline
- `U+00A0` - Non-breaking space

Parse unicode ranges `glyphhanger https://deploy-preview-28--skok.netlify.com/blog/unicode-ranges --spider --spider-limit=100`

```sh
pyftsubset Inter-VarRoman.otf \
  --unicodes="U+20-7E,U+00A9,U+2026,U+A,U+00A0" \
  --layout-features="calt,ss01,ss02,zero" \
  --flavor="woff2" \
  --output-file="Inter-VarRoman-subset.woff2"
```

## Contextual alternates

### `calt`

Disabled: <p>M@N m@n</p>
Enabled: <p style="font-feature-settings: 'calt' 1">M@N m@n</p>

## Case Alternative

Disabled: <p>'3 + 9 = 12 \* 1'</p>

Enabled: <p style="font-feature-settings: 'case' 1">'3 + 9 = 12 \* 1'</p>

## Stylistic set 1: Alternate digits

### `ss01`

Disabled: <p>13469</p>
Enabled: <p style="font-feature-settings: 'ss01' 1">13469</p>

## Stylistic set 2: Disambiguation

### `ss02`

Disabled: <p>WP0ACO9XSI</p>
Enabled: <p style="font-feature-settings: 'ss02' 1">WP0ACO9XSI</p>

## Slashed Zero

### `zero`

Disabled: <p>O0123</p>
Enabled: <p style="font-feature-settings: 'zero'">O0123</p>
